5) Branch

git checkout -b test1 - создает ветку и переключается на нее

git branch -  посмотреть ветки

git branch test2 -  создает ветку

git branch -d test2 - удалить ветку

git checkout test2 - переключение на нужную ветку

git branch -v  - посмотреть все ветки

git config --global merge.tool kdiff3  - казывает утилиту для мержа

git mergetool -запуск утилиты
